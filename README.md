# BRÚJULA | Comunicación Estratégica

Web institucional responsive para BRÚJULA, una consultora de comunicación estratégica que diseña rumbo, herramientas y acompañamiento para que organizaciones comuniquen con autonomía.

## Archivos

- `index.html`: estructura principal de la web.
- `styles.css`: identidad visual, responsive y animaciones.
- `script.js`: menú móvil y animaciones sutiles.
- `assets/brujula-hero.svg`: recurso visual editable.
- `brujula-post-redes-copy.md`: texto sugerido para publicación en redes.
